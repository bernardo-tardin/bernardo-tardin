# 🏎️ Bernardo Moraes
**`Biomedical Engineer building at the intersection of ML and real-world data systems`**

MSc in Biomedical Engineering (Medical Informatics) with a background in backend development and a growing focus on Machine Learning. I build end-to-end — from 
messy raw data to trained models to clean, maintainable code.

My dissertation was a full clinical data platform (Django · Docker · Redis · RabbitMQ).  My current focus is ML: I've been working through a structured portfolio of projects  spanning NLP and Computer Vision, progressively tackling more complex architectures.
